Harambee Dinner Programs and Analysis

Even though the Harambee dinner has been held annually for more than 30 years, only eight dinner programs existed in the Penn State Behrend Archives in 2018. These programs have been processed and you can click the links below to view them. These programs were placed at each seat for guests to view the schedule of events, the menu, and a list of the individuals and clubs that made the event possible. Both the page and the text will be displayed side-by-side for you to explore the history of the Harambee dinner.

Analysis

There are many interesting questions that can be answered by the data provided in these programs. Who participated in these events the most and in which years? Who was thanked the most? Which clubs have participated every year? Here you can see a visual representation of some of these answers. If you would like to view the code yourself, visit the GitHub Repo!
